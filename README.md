# SocialSpark

## Project Description

Social Spark is an Android application developed using Kotlin in Android Studio.
The app provides users with helpful and motivational suggestions based on the time of day they enter. It demonstrates basic programming concepts such as conditional logic, user input validation, and interactive mobile interface design.

---

## Purpose of the App

The purpose of the Social Spark application is to:

* Provide motivational or social suggestions to users.
* Demonstrate the use of IF statements in Kotlin.
* Allow users to interact with a simple Android application.
* Show how input validation improves user experience.

This project was created as part of an assignment to demonstrate mobile development and the use of version control.

---

## Features

* User input field for time of day.
* Smart validation of user input.
* Different suggestions based on:

  * Morning
  * Afternoon
  * Evening
* Error messages for incorrect entries.
* Clean and simple interface.

---

## How the App Works

1. The user opens the app.
2. The user enters a time of day.
3. The app checks the input.
4. The app displays a relevant suggestion or message.

Example logic used:

* Morning → Motivational message
* Afternoon → Productivity suggestion
* Evening → Reflection message
* Invalid input → Error message displayed

---

## Design Considerations

The following design principles were used when creating the app:

### Simplicity

The application was designed with a simple layout to make it easy for users to understand and interact with the app.

### User Experience

Clear instructions and feedback messages were included to help users know what input is required.

### Error Handling

Input validation ensures that incorrect or empty entries are handled properly.

### Readability

The interface layout and text were structured so that users can quickly read suggestions and responses.
